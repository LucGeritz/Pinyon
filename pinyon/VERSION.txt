pinWebApp
---------
Ctrl.onBegin passes action
Ctrl.onEnd passes action
New pseudo-event: WebApp.onCanceled, this is called when ctrl.onBegin
returns false. As parameter it passes the controller.
Default implementation of onCanceled does nothing..

pinBaseController
-----------------
onBegin needs parameter action (default '')
onEnd needs parameter action (default '')

pinRouter
---------
new setting: ipclosedexemption. 
  This ip address is allowed to start routes even 
  when the site is closed ( isclosed in pinWebApp
  set to true). Basicly  now you  can  close  the 
  site "for the rest of the world".
